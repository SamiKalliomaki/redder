# Redder

Redis-compatible implementation in Rust.

This has been made for leaning purposes, and it's not intended to be used in production.

## Features

### Supported commands
 - [x] PING
 - [x] ECHO
 - [x] GET
 - [x] SET
 - [x] GET CONFIG
 - [ ] Clear memory on key expiry

### RDB
 - [x] Read RDB file
 - [ ] Write RDB file

### Data types
 - [x] String
 - [ ] Anything else
